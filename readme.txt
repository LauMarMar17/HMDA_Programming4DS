FOLDERS AND FILES:

* Data_visualization folder contains the data_visualization.ipynb with the second part of the assignment. 
    It contains the data transformation and manipulation process as well as the plots answering the questions.

* Outputs folder contains all csv files obtained from the Scrapping process (awards, new_movies_DE, new_movies_ES
    new_movies_GB, new_movies_IT, top_250) and some csv after the transformation/manipulation process 
    (new_movies_cleaned, top_250_cleaned)

* Web_scrapping folder contains the data_scrapping.ipynb with the main code of the scrapping process, the executor
    msedgedriver.exe to be able to connect to the browser, and a useful.py with some important functions used in the 
    scrapping process.
